FINDER: Gateway Project Overview

Bem-vindo. Abaixo estão as instruções para salvar seu projeto no Finder.

Campos marcados com * são obrigatórios. Portanto, é obrigatório preencher: Name Project, Owner e Status.

Os demais campos podem ser preenchidos posteriormente.

Você tem 15 dias para completar as informações dos servidores. Caso os campos de servidores permaneçam vazios após esse prazo, o projeto será excluído do Finder, sendo necessário realizar um novo registro.
